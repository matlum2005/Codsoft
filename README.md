# Codsoft Full-Stack Java Projects

Welcome to my **Codsoft Internship** project repository! This repository contains a collection of Java-based projects ranging from simple terminal games to full-fledged management systems with Graphical User Interfaces (GUIs).

## 🚀 Projects Overview

This repository is organized into 5 primary tasks, each demonstrating different aspects of Java development, including core logic, object-oriented programming, and GUI design.

---

### 🎮 Task 1: Number Guessing Game
A fun and interactive CLI-based game where the user tries to guess a randomly generated number within a limited number of attempts.
- **Key Features**: Random number generation, score tracking, multiple rounds, and attempt limits.
- **Tech Stack**: Core Java.

### 📊 Task 2: Student Grade Calculator
A utility program that calculates a student's total marks, average percentage, and assigns a grade based on their performance across multiple subjects.
- **Key Features**: Input validation, percentage calculation, and automated grading system.
- **Tech Stack**: Java (Logic).

### 🏦 Task 3: ATM Machine Interface
A robust simulation of an Automated Teller Machine (ATM) that interacts with a user's bank account.
- **Key Features**: Withdrawals, Deposits, Balance Checks, and Transaction Validation.
- **Tech Stack**: Java (OOP principles).

### 📝 Task 4: Quiz Application with Timer
An engaging quiz application that presents multiple-choice questions with a countdown timer for each.
- **Key Features**: Timer integration, automated scoring, and result summary.
- **Tech Stack**: Java, Maven.

### 🎓 Task 5: Student Management System
A comprehensive desktop application designed to manage student records with a professional user interface.
- **Key Features**: Add/Edit/Remove students, Search functionality, Persistence, and a JavaFX-powered GUI.
- **Tech Stack**: Java 17, JavaFX, Maven, Spring Boot (Backend).

## 🛠️ Getting Started

### Prerequisites
- **Java Development Kit (JDK) 17** or higher.
- **Apache Maven** installed.
- An IDE like **IntelliJ IDEA**, **Eclipse**, or **VS Code**.

### Installation & Execution

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/matlum2005/Codsoft.git
   cd Codsoft
   ```

2. **Build a Task (Example: Task 5)**:
   ```bash
   cd "Task 5/student-management-system"
   mvn clean install
   ```

3. **Run the Application**:
   Each folder contains its own execution logic. For Task 5 (JavaFX):
   ```bash
   mvn javafx:run
   ```
